Color Palette for AgroGest

This modern color palette is inspired by natural elements of agricultural fields: greens for vegetation, blues for water, and amber for sun/harvest.

Primary Colors

These colors represent the main brand identity of AgroGest.

Primary
Main Green
--primary: #2E7D32

Main application color, represents nature and agriculture.

Primary Light
Light Green
--primary-light: #60AD5E

Light variant of the main color, ideal for backgrounds and secondary elements.

Primary Dark
Dark Green
--primary-dark: #005005
--bs-primary: #2E7D32

Dark variant for hover states and highlighted elements.

Secondary Colors

These blues complement the greens and represent water/irrigation.

Secondary
Blue
--secondary: #1565C0
--bs-secondary: #1565C0

Secondary color, ideal for alternative actions and irrigation.

Secondary Light
Light Blue
--secondary-light: #5E92F3

Light variant of blue, for backgrounds and subtle elements.

Secondary Dark
Dark Blue
--secondary-dark: #003C8F

Intense blue for active states and hover.

Bootstrap Integration

To use this palette with Bootstrap, you can override Bootstrap's variables in your SCSS:

$primary: #2E7D32;
$secondary: #1565C0;
$success: #4CAF50;
$warning: #FF9800;
$danger: #F44336;
$info: #2196F3;

$light: #F5F7FA;
$dark: #1F2933;

// Then import Bootstrap
@import "bootstrap/scss/bootstrap";
